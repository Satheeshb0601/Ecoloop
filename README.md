# EcoLoop 🌿

**EcoLoop** is a modern, premium smart waste management single-page application (SPA) designed to help users dispose of household waste responsibly and track their environmental footprint.

## ✨ Features

- 🌌 **Premium Glassmorphic UI**: Clean translucent cards with dynamic mouse-tracking spotlight borders.
- 🧬 **Interactive Node Particle Background**: A responsive canvas network that reacts dynamically to mouse movement.
- 🌋 **Liquid Ambient Lights**: Floating glowing fluid light sources using an SVG gooey filter (`#goo`) for a high-end look.
- 💡 **Waste Disposal Rules Engine**: Get instant segregation rules, decomposition timelines, and actionable disposal steps for Organic, Plastics, Paper, Metal, Glass, E-Waste, and Clothes.
- 🛠️ **DIY Upcycling Cards**: Discover 4 creative upcycling ideas for every waste category to repurpose items at home.
- 📊 **Impact Dashboard**: Log waste disposal events to track total kilograms of waste saved, CO₂ emissions prevented, and increase your Eco Rank.

## 🌐 Live Demo

You can view the live, fully interactive application hosted on GitHub Pages:
👉 **[https://xoxosuriya.github.io/EcoLoop](https://xoxosuriya.github.io/EcoLoop/)**

### How to enable hosting on GitHub:
1. Go to your repository settings page on GitHub.
2. Click on **Pages** in the left sidebar under the "Code and automation" section.
3. Under **Build and deployment**, select **Deploy from a branch** as the source.
4. Set the branch to `main` and directory to `/ (root)`.
5. Click **Save**. Within a minute, your project will be live!
